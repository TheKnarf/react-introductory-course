Exercise 1 - Setting up webpack and Babel

Step 1 - Setting up an empty webpack

Webpack is used for bundeling all your source files into a single javascript bundle. That way you wont have to include each one of them manually in the HTML file.

By default webpack tries to parse the file ./src/index.js . So start of by creating an empty folder and then a file called index.js . 

npm i --save-dev webpack webpack-cli

Step 2 - html-webpack-plugin

html-webpack-plugin 
npm i --save-dev html-webpack-plugin

Step 3 - Setting up babel

Instructions taken from https://babeljs.io/. Always use the webpage for updated documentation

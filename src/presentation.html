 What is React? 

						
React is a declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

 Babel 

						 https://babeljs.io 
						
 Transpiling modern JavaScript and JSX into plain code. 
   
					

					

						
 Babel gets configured with plugins to compile code 

						
 For React we want two presets: 

						
npm i --save-dev @babel/preset-env @babel/preset-react

					

					

						
 Then we can configure babel by making a ".babelrc" file: 


{
  "presets": ["@babel/preset-env", "@babel/preset-react"]
}

 Loaders! 

					

					

						
 A loader is simply a function that takes some code inn and returns a processed version 


module.exports = source => {
   return source
      .replace(/\<code\>(([\n]|.)+?)\<\/code\>/gi,
         (matches, g1) => {
   const code = g1.replace(//g, '>');
   return `<code>${code}</code>`;
});
